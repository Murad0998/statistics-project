Bank Marketing Analysis

Exploratory Data Analysis & Statistical Hypotheses Testing

 Описание проекта:

Проект посвящён анализу маркетинговой кампании банка, направленной на привлечение клиентов к открытию срочного депозита.
Цель проекта — с помощью EDA и статистических гипотез выявить факторы, влияющие на отклик клиентов, и дать интерпретируемые бизнес-выводы.

Анализ выполнен на основе открытого датасета Bank Marketing Dataset (UCI ML Repository).

------------------------------

Цели и задачи:
	-	Провести разведочный анализ данных (EDA)
	-	Подготовить данные к анализу (очистка, типы, бинаризация целевой переменной)
	-	Сформулировать проверяемые статистические гипотезы
	-	Проверить гипотезы с использованием подходящих статистических тестов
	-	Оценить величину эффекта, а не только статистическую значимость
	-	Сформулировать практические рекомендации для бизнеса

Описание этапов

1 Exploratory Data Analysis (01_eda.ipynb)

На этапе EDA:
	-	изучена структура датасета (41 188 наблюдений, 23 признака)
	-	выявлены и проанализированы пропущенные значения
	-	заменены значения "unknown" на NaN
	-	приведены типы данных (category / int / float)
	-	исследованы распределения числовых признаков
	-	выявлен дисбаланс классов
	-	проведён первичный анализ зависимостей (визуально и корреляционно)

EDA используется для генерации гипотез, а не для принятия окончательных решений.

------------------------------

2 Гипотезы для категориальных признаков (02_hypotheses_categorical.ipynb)

Проверялись гипотезы о различии долей отклика (y=1) между группами, сформированными по категориальным признакам, в том числе:
	-	уровень образования
	-	профессия
	-	тип контакта
	-	семейное положение
	-	наличие кредитов
	-	результат предыдущей кампании (poutcome)
	-	наличие предыдущих контактов (previous > 0)

Используемые методы:
	•	χ²-тест независимости
	•	Z-тест для сравнения долей
	•	Bootstrap-оценка доверительных интервалов
	•	Cramér’s V и Risk Ratio для оценки эффекта

------------------------------

3 Гипотезы для числовых признаков (03_hypotheses_numerical.ipynb)

Анализировались различия числовых признаков между группами отклика:
	-	длительность звонка (duration)
	-	количество контактов (campaign)
	-	время с последнего контакта (pdays)
	-	макроэкономические показатели

Методы:
	-	Mann–Whitney U test
	-	Bootstrap для оценки разницы медиан
	-	анализ взаимодействий

------------------------------

Бизнес-выводы
	-	Приоритизировать «тёплых» клиентов с историей контактов
	-	Использовать двухэтапные кампании (прогрев → конверсия)
	-	Делать упор на мобильный канал связи
	-	Не ориентироваться только на p-value при принятии решений
